## Getting Started

Using biwa scheme

<https://github.com/biwascheme/biwascheme/blob/master/README.md>

### How to use with node.js to run a biwa script

--------------------------------------------

1. $ npm install biwascheme
2. create a file a.scm:
```
  (display "Hello, world!")
  (newline)
```
3. $ biwas a.scm
